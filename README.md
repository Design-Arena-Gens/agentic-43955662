## Shorts Architect

AI-assisted blueprint generator for shipping high-retention YouTube Shorts. Drop in a topic, pick a tone and pacing, and instantly get a hook, beat-by-beat voiceover script, editing cues, captions, and optional AI asset prompts.

### Features

- Tone aware hooks and narrative beats tuned for Shorts retention
- Beat grid with timestamps, VO lines, on-screen text, and visual cues
- Editing room directives covering transitions, sound design, and visual anchors
- Caption + hashtag package ready for publishing
- Optional AI prompts for b-roll, thumbnails, and voiceover synthesis

### Local Development

```bash
npm install
npm run dev
```

### Build & Lint

```bash
npm run lint
npm run build
```

### Deploy

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-43955662
```
